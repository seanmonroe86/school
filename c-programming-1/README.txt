Compile the source code file 'temp.c' using either gcc or cc.
Execute 'temp.out' with './'

Example:

gcc -o temp.out temp.c
./temp.out

or

cc -o temp.out temp.c
./temp.out

Note:
All instances of 'temp.out' in this README may be replaced with
another filename, such as 'temp.o'.
